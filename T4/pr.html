Abstract:

No trabalho prático nº4, o primeiro objetivo passou por criar uma stylesheet para, a partir do dataset do Arquivo Sonoro, gerar páginas xml individuais para cada documento. Estas páginas foram nomeadas com docX.html, onde X representa um id atribuído ao documento, que irá ser usado no servidor.

Após gerar essas páginas, foi pedido para gerar um xsl( doc2html.xsl) para converter os ficheiros xml de cada documento em páginas html.

Após criar essa stylesheet passamos para a criação do servidor em Node.js.

Este servidor recebe pedidos de GET em "/musica/X", e abre uma página HTML com o ficheiro docX.xml já convertido em HTML. Todos os outros pedidos são ignorados pelo servidor, apresentando apenas uma mensagem de erro.
