Abstract:

No trabalho prático nº7 foi pedida a implementação de uma API de dados para executar pedidos a uma base de dados de Obras Musicais.

Essa API teria que ser capaz de excutar pedidos em '/obras' e em '/compositores'. Assim sendo, foram criados 2 routers distintos para executar os pedidos de cada rota.

Para os diferentes pedidos foram utilizadas querys e parâmetros URL para enviar as informações de filtragem necessárias para a BD.

A base de dados utilizada foi o MongoDB.
