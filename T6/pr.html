Abstract:

O objetivo do sexto trabalho prático de PRI era a criação de um servidor para interagir com uma base de dados em Mongo e que fosse capaz listar, adicionar ou remover informações dessa base de dados.

A base de dados disponibilizada para esta resolução contém informações acerca do título, ano de realização, actores e géneros de mais de 25 mil filmes.
