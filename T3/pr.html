Abstract:

Este terceiro trabalho prático consiste em aplicar os conhecimentos adquiridos no decorrer das aulas da UC para criar dois ficheiros '.xsl'. O primeiro origina uma página html para este Project Record e o segundo, a partir do ficheiro '.xml' dos Arqueossítios, gera uma página 'index.html' com o índice de todos os arqueossítios e, para cada arqueossítio, gera a sua própria página html com a informação relativa a ele próprio.

Deliverables:

  • Página HTML - Manifesto
  • Ficheiro DTD - Manifesto
  • Stylesheet - Manifesto
  • Documento XML - Manifesto
  • Página HTML - Índice dos Arqueossítios
  • Dataset dos Arqueossítios (XML)
